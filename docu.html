Introduction

The purpose of this webpage is to allow players of Final Fantasy XIV to search the game's database so that they can find basic information on different parts of the game. This could range from items, to enemies, to player moves, the reason why a simple search is important is due to the eubiquity of many of the images. Regardless of what language you are playing in, the icons for these aspects are all the same, allowing the user of this app to look for the image if needed. They also have the ability to save to favorites so they can add their preference to the cloud to display for others.

CSS Framework

CSS Framework provided by Bulma (https://bulma.io/), icons provided by Square Enix CO. LTD.

API Information

Final Fantasy XIV API website: https://xivapi.com/, full documentation: https://xivapi.com/docs
This API keeps track of all of the entities that currently exist within the game of Final Fantasy XIV, which was created by Square Enix

Functions

loadJsonFetch()

This function loads information from the FFXIV API when the user searches for something then displays it on screen

saveData()

This function saves the results that have been clicked to local storage so that they can be seen later

clearData()

This function clears the local storage and favorites screen

loadData()

This function reads the local storage and then displays the favorites on the favorites page

downloadFaves()

This function reads the linked Firebase databsese and displays favorites from teh page with the amount of likes that have been accumulated on the favorites page

writeFavNameData()

This function updates the Firebase database so that new functions can be added, and if something is present, the like count is updated

exportDownFaves()

This function helps run downloadFaves()
