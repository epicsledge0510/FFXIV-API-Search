Welcome to my Final Fantasy XIV database search site! This site allows you to search for objects inside of the API and game for Final Fantasy XIV Online.

The "Search" tab will allow you to search the database while focusing on the type of item, action, etc. as well as limiting the amount. Clicking on an icon will add it to your favorites, which you can view or clear under the "Favorites" tab.
